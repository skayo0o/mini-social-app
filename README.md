# Facepoop - Мини-социальное приложение

Социальная сеть на React для обмена постами, лайков и редактирования. Создано в образовательных целях для изучения React, JSX и управления состоянием.

## Основные возможности

- **Создание постов** - публикуй свои посты с никнеймом
- **Редактирование** - исправляй свои посты
- **Лайки** - один лайк на пост от пользователя
- **Удаление** - удаляй свои посты (только автор)
- **Анимация печатания** - начните вводить текст...
- **Пасхалка** - попробуйте написать "казино" в теле поста...
- **Система никнеймов**

### Установка

# Устанавливаем зависимости
npm install

# Запускаем dev сервер
npm run dev
```
Приложение будет доступно на `http://localhost:5173`

### Обязательные требования

| Требование | Реализация | Соответсвие |
|-----------|-----------|---------|
| **Компоненты** | 6 компонентов с разделением ответственности | Присутсвует |
| **Props** | Передача данных между компонентами | Присутсвует |
| **State** | useState для управления состоянием | Присутсвует |
| **События** | Click, Change, Submit обработчики | Присутсвует |
| **Списки** | map() для отображения элементов | Присутсвует |
| **Keys** | Уникальные ключи для элементов списка | Присутсвует |
| **Условный рендеринг** | Ternary операторы и &&-операторы | Присутсвует |
